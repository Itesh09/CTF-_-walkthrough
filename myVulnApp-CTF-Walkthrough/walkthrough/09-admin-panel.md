# 09 - Admin Panel

## Objective
Access the admin panel and find the final flags.

## Methods to Access

1. Use credentials found earlier
2. Cookie manipulation
3. Force browsing to admin URLs

## Common Admin Paths

- `/admin`
- `/administrator`
- `/adminpanel`
- `/manage`
- `/backend`

## Post-Exploitation

Once authenticated as admin:
- Check admin dashboard
- Look for flag files
- Examine user management

## Flag

Gain admin access and find the admin flag.
